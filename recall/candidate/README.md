## Usage

### 1. Get valid candidate

Run script:
```shell
sh ./get_valid_candidate.sh DATA_PATH SAVA_FILE E2R_SCORES_FILE
```

### 2. Get test challenge candidate
Run script:
```shell
sh ./get_test_candidate.sh DATA_PATH SAVA_FILE E2R_SCORES_FILE
```
